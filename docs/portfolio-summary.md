# Portfolio Summary

## One-Line Pitch

Smart Academic Scheduler is a full-stack academic operations platform for universities that combines scheduling algorithms, institutional constraints, analytics, and AI-ready explanations.

## Strong Interview Talking Points

- Migrated the frontend toward `Next.js + TypeScript + Tailwind`
- Preserved the same deployment-link migration path for `Vercel` and `Render`
- Used `heap` and `dynamic programming` in a real scheduling problem
- Added teacher availability, room blackout windows, locked sessions, batch conflict prevention, and room feature constraints
- Built assistant-style explanations and suggestions on structured scheduling data
- Added analytics for room utilization, teacher load, and schedule quality

## Best Demo Flow

1. Show dashboard and product framing
2. Open setup pages for rooms/courses
3. Open scheduler run page and explain constraints
4. Open conflict center
5. Open analytics
6. Open AI assistant and explain the recommendation flow

## Why Recruiters Should Care

- It is not a toy CRUD app
- It has genuine algorithmic thinking
- It has product thinking for real institutions
- It shows modern frontend stack alignment
- It demonstrates AI integration in a practical workflow
